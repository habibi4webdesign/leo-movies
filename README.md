# Installation Instructions

```
$ git clone https://github.com/habibi4webdesign/leo-movies
```

then intall all dependencies:

```
$ cd leo-movies
$ yarn

```

it's ready to run by this command:

```
$ yarn start
```

for runnig test use this command:

```
$ yarn test
```
